# 표준규격의 "층위와 국적" — 어떤 규격이 어디 소속인가

> K-METCAL 사업 준비용. 규격 접두어만 봐도 소속·적용범위·수출 함의를 안다.
> 작성일: 2026-06-16. 근거: MIL-STD-1839D 표지 직접 확인 + KAI 사업기획서 p.26~27.

---

## 0. 핵심 — 규격에도 "국적/소속"이 있다 (접두어로 구분)

| 접두어 | 발급 주체 | 적용 범위 | 성격 |
|---|---|---|---|
| **MIL-STD / MIL-PRF / MIL-HDBK** | 미국 **국방부(DoD)** | 美 전군(육·해·공·해병) | 군 규격 |
| **DAFMAN / AFMAN / TO** | 미 **공군** | 공군(+우주군)만 | 공군 내부 규정 |
| **ANSI/NCSL** | 미국 **민간** (ANSI 표준협회 + NCSL 계측학회) | 민간·정부 공통 | 미국 민간 표준 |
| **ISO / IEC** | **국제** 표준기구 | 전 세계 | 국제 표준 |
| **DAFPD / AFPD** | 미 공군 (정책) | 공군(+우주군) | 정책 디렉티브 |
| **NAVAIR / OPNAVINST / SECNAVINST** | 미 **해군** | 해군 | 해군 규정 |
| **STANAG / AQAP** | **NATO** | NATO 32개국 | 北大西洋조약기구 표준 |
| **DI-QCIC / DID** | 미 국방부 | 군 (데이터 납품) | 데이터 항목 기술서(양식) |

## 1. 층위 그림 — "특수규격"이 아니라 "층위가 다른 표준들"

```
국제      ISO/IEC 17025, ISO 10012        ← 전 세계 공통 (가장 넓음, 수출 유리)
  │
미국민간  ANSI/NCSL Z540.3                 ← 미국 민간 국가표준 (군이 계약서에 인용)
  │
미 국방부  MIL-STD-1839D, MIL-PRF-38793C   ← 美 전군 공통 (공군 AFMETCAL이 대표 관리)
  │
미 공군   DAFMAN 21-113, TO 00-20-14      ← 공군(+우주군) 전용 (가장 좁음)
```

---

## 2. 자주 헷갈리는 2개 — 정확히

### MIL-STD-1839D = 미 공군 전용 ❌ / 美 전군(DoD) 공통 ⭕
- 표지 증거: "DEPARTMENT OF **DEFENSE** STANDARD PRACTICE" / "all Departments and Agencies of the DoD" / "DISTRIBUTION STATEMENT A. public release"
- 즉 육·해·공·해병 공통. 단 **관리사무실이 공군 AFMETCAL(Heath, OH)** (MIL-PRF@AFMETCAL.AF.MIL).
- → "공군 전용"이 아니라 **"공군이 펜을 쥔 전군 규격"**. 그래서 사업기획서 p.26 "양군(공군·해군) 공통 DoD 기반"에 등장.
- 내용: 교정·측정 요구사항 (TUR, NIST 소급성, 교정주기). "NOT MEASUREMENT SENSITIVE"=내용이 측정값에 민감하지 않다는 행정표시, 비밀 아님.

### ANSI/NCSL Z540.3 = 군 규격 ❌ / 미국 민간 표준 ⭕
- ANSI = American National Standards Institute (미국 표준협회, 민간)
- NCSL = National Conference of Standards Laboratories (계측 표준 실험실 학회)
- 즉 **민간이 만든 미국 국가표준**. 군이 **계약서에 "이 표준 따르라"고 인용**해서 군 사업에 등장 (기획서 "군 계약 인용/필수").
- 내용: 교정 결과의 **통계적 신뢰성** — TUR ≥ 4:1, False Accept ≤ 2%.

## 3. ★ K-METCAL 수출 전략 함의 (가장 중요)

> **위 층위로 갈수록(국제 ISO) "전 세계 공통"이라 수출에 유리. 아래(미 공군 전용)만 베끼면 미국 밖엔 못 판다.**

- KAI가 "NIST·KRISS·NATO 3중 표준 정합성, Vendor-Neutral"(p.15)을 강조하는 이유 = 넓은 층위를 잡아야 NATO 32개국에 판매 가능.
- 폴란드·말레이시아 확장 논리의 뿌리. ISO 17025 + STANAG/AQAP를 잡으면 NATO 자동 상호인정(STANAG 4107).
- 결론: 군 전용 규정(DAFMAN/TO)은 "참조 모델"로, **국제·전군 규격(ISO/MIL-STD)은 "제품 정합성"으로** 잡는다.
