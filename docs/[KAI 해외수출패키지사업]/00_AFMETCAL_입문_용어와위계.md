# AFMETCAL 입문 — 군 표준문서 용어 · 위계 · 핵심개념

> K-METCAL 사업 준비용 기준 문서. 다른 미 공군 표준문서를 읽을 때 계속 참조한다.
> 1차 근거: **DAFMAN 21-113 (27 Dec 2023, 29쪽)** 전문 정독.
> 작성일: 2026-06-16

---

## 0. 한눈 요약

> **AFMETCAL = 미 공군의 모든 측정장비(TMDE)가 '안전·정확·균일·신뢰·소급성'을 갖추도록,
> 전 세계 실험실(PMEL)을 계층적으로 묶어 NIST(국가표준)까지 끊김 없이 연결하고,
> 누가 무엇을 책임지는지 명확히 정한 시스템.**

- **정신(철학·조직·책임)** = `DAFMAN 21-113` ← 우리가 읽은 문서
- **손발(실제 작업 절차)** = `TO 00-20-14` ← 아직 미입수, 입수 필요
- 시스템의 척추 = **소급성 체인(Traceability)**

## 1. 표준 명칭 읽는 법 — "택배 주소"처럼 3부분으로 분해

### (1) 알파벳 = 문서의 '급(級)' = 강제력의 종류

| 약어 | 풀이 | 한국 법 비유 | 역할 |
|------|------|--------------|------|
| **PD** (Policy Directive) | 정책 지침 | 헌법/법률 | "우리는 ~를 한다" 의지 선언 (짧고 추상) |
| **MAN** (Manual) | 매뉴얼 | 시행령 | 정책을 "이렇게 운영한다" — 조직·역할·책임 |
| **I** (Instruction) | 지시 | 시행규칙 | 야전 부대가 지키는 현장 규칙 |
| **TO** (Technical Order) | 기술교범 | 작업표준서(SOP) | 손으로 따라 하는 실제 절차 (How-to) |

### (2) 접두어 = '누가 만들었나' — AFMAN vs DAFMAN의 핵심 차이

- **AF** = Air Force (공군만)
- **DAF** = Department of the Air Force (공군**부** = 공군 + **우주군 USSF**)

> **AFMAN → DAFMAN 변경 = 2019년 우주군 창설로, 공군 전용 문서를 공군+우주군 공용으로 전환한 것.**
> 근거(추측 아님): DAFMAN 21-113 1쪽 SUMMARY OF CHANGES —
> "converting to DAFMAN for applicability for both USAF and USSF."
> → 첨부 이미지의 `AFMAN 21-113 → DAFMAN 21-113` 화살표는 **같은 문서의 옛 이름→새 이름**.
> (폴더의 두 PDF는 304KB로 동일 파일)

### (3) 숫자 = '주제 영역(시리즈)' + 일련번호

| 시리즈 | 주제 |
|--------|------|
| **21** | 정비(Maintenance) ← 교정도 정비의 일부. 우리 영역 |
| 33 | 통신·기술교범 (33K = 교정 절차 TO) |
| 36 | 인사·교육 |
| 00- | 전군 공통 기술교범 |

### 예시 해독
- **DAFMAN 21-113** = 「공군부(DAF) / 매뉴얼(시행령급) / 정비(21) / 113번」
  = 공군·우주군 교정 프로그램 운영 시행령
- **DAFI 21-101** = 「공군부 / 지시(시행규칙) / 정비(21) / 101번」 = 항공기·장비 정비 관리 현장 규칙

## 2. 문서 위계도 (첨부 이미지 = 이 구조)

```
DAFPD 21-1  「군 장비를 정비한다」                         ← 헌법 (의지)
  │ implements
  ├─ ★ DAFMAN 21-113  교정 프로그램 운영 골격(조직·역할·철학) ← 시행령 ★우리가 읽음
  │     │ "어떻게(How)"는 전부 아래로 위임 ("in accordance with TO 00-20-14" 수십 회)
  │     └─ TO 00-20-14  AF Metrology/Calibration Program        ← 실제 절차서(SOP) ★미입수
  │          + TO 33K-1-100-1 / -2-1 (교정주기·CMS 데이터)
  └─ DAFI 21-101  항공기·장비 정비 관리                        ← 시행규칙(야전 현장)
```

핵심: **21-113은 "무엇을·왜·누가"만 말하고 "어떻게"는 TO 00-20-14에 위임한다.**

### ★★ KOLAS 출신 필수 개념 — "교정 = 정비(maintenance)의 일부" (KAI 프로젝트 핵심)
- **PMEL은 교정만 하는 곳이 아니다.** TO 00-20-14 §1.4.21 정의: PMEL은 TMDE를 **maintain
  = troubleshoot(진단)+align(조정)+repair(수리)+calibrate(교정)** 한다. 교정은 4개 중 하나.
- **KOLAS vs 군**: KOLAS 교정소는 부정확하면 "부적합→반송"(수리는 제조사). 군 PMEL은
  **직접 진단·수리·교정까지 원스톱**. 전장에선 제조사 보낼 시간 없어서. → PMEL=계측기 전문 정비소.
- 그래서 PMEL은 **정비부대 규칙(DAFI 21-101)**도 따름: **FOD**(이물질손상방지·출장 시 공구검수),
  **CTK**(공구키트 재물조사), **MSEP**(정비표준화평가). KOLAS엔 없는 개념.
- **DAFPD 21-1이 "교정은 정비의 하위"라 한 게 이 의미.** 노란색 형제문서 DAFI 21-101(정비관리)·
  DAFI 20-101/63-101(수명주기)이 21-113과 같은 정책 아래 있는 이유.
- **사업 시사**: PAMS-K는 교정 이력만 아니라 **진단·수리·정렬 이력**까지 관리 대상일 수 있음.
- **자료 분기**: 이 "정비 통합" 측면은 군색 짙음 → **KAI엔 필수, ICMPM(국제표준 ISO17025)엔 무관·배제**.

## 3. 핵심 용어 5개 (이것만 외우면 90% 이해)

| 약어 | 우리말 | 무엇인가 | 비유 |
|------|--------|----------|------|
| **TMDE** | 측정·시험·진단 장비 | 교정 대상 모든 계측기(토크렌치·압력계·멀티미터…) | 환자 |
| **PMEL** | 정밀측정장비 실험실 | TMDE를 교정·인증하는 기지 단위 실험실 | 병원 |
| **AFPSL** | 공군 1차표준실험실 | 美 공군 **유일 최상위** 표준실 (오하이오 Heath). NIST 직결 | 대학병원 본원 |
| **AFMETCAL** | 프로그램명 + 운영조직(AFLCMC/WNM) | 전체를 기획·예산·인증·기술권한 총괄 | 보건복지부 |
| **NIST** | 미 국립표준기술원 | 모든 소급성의 종착점 (한국 KRISS에 해당) | WHO |

### 소급성 체인 (시스템의 척추)
```
NIST ─→ AFPSL ─→ (Base Std) ─→ PMEL ─→ (Working Std) ─→ 야전 TMDE
국가표준  공군표준    기지표준        실험실   작업표준          실제 무기 계측기
```
- "끊김 없는 비교의 사슬(unbroken chain of comparisons) + 명시된 불확도"
- → 임채욱 KTL 선임 ICMPM 키노트 "DCC in Industrial Field"와 동일 줄기

## 4. AFMETCAL의 헌법 제1조 (DAFMAN 21-113 §1.1)

> measurements are **safe, accurate, uniform, reliable, and traceable**
> to the International System of Units through **NIST** or AFMETCAL approved sources.

이 5개 단어(안전·정확·균일·신뢰·소급)가 정신의 핵심. 나머지 28쪽은 전부 이걸 지키는 장치.

## 6. ★ K-METCAL 관점 — 가장 눈여겨볼 4가지

### ① 교정주기 신뢰성 85% 규칙 (§2.6.14.2) — 최우선
> 교정주기는 통계 분석으로 "주기 말 신뢰성(end-of-period reliability) 85% 이상"이 되도록 조정.
- CaliBoard 교정주기 분석 탭(Phase G)과 **동일 문제**.
- 우리 "유사장비 빅데이터 교정주기 예측"의 **공식 정당성 근거**.

### ② 건강한 네트워크 3대 지표 (§2.14.5) — 대시보드 KPI
- 가용률 **92%** / 회송시간(Turnaround) **<15일** / 처리시간(Throughput) **<30일**
- K-METCAL 대시보드 핵심 KPI 후보. CaliBoard reception·freshness가 유사 기능 보유.

### ③ PAMS / MetWeb® — 그들의 기존 시스템 = 우리 벤치마킹/대체 대상
- **PAMS** = PMEL Automated Management System (미 공군 교정 관리 시스템)
- **MetWeb®** = 메트롤로지 데이터 웹앱
- → K-METCAL = 본질적으로 "PAMS 한국형" 구축 사업. 기능 명세의 출발점.

### ④ 인증(Certification) 권한 구조 — 진입장벽 + 권위
- AFMETCAL Director가 PMEL을 인증/인증취소하는 단일 권위 (§2.6.16, 2.6.25~27)
- 소급성·기술능력·시설 미달 시 인증 박탈 → 시스템에 "인증 상태 관리" 핵심 기능 필요 시사.

## 7. 그 밖에 알아둘 개념

- **(T-0 ~ T-3)**: 각 규정 뒤 붙는 면제(waiver) 권한 등급. T-0=가장 엄격(외부 법령 근거), T-3=현장 지휘관 수준.
- **TMDE 한계교정(Limitation, §1.4)**: 토크렌치·압력계 등 범용 장비는 사용자 승인 하 "제한적 교정" 가능. (토크렌치가 문서에 직접 예시로 등장)
- **CMS** (Calibration and Measurement Summary): 특정 무기체계의 교정 개념·권위를 담은 TO.
- **MCF** (Metrology and Calibration Flight): 창정비(Depot, ALC)용 현장 교정대. PMEL의 창정비판.
- **RNI/RNM** (Repair Network): 교정을 "수리 네트워크"로 보고 노드 단위로 관리하는 개념.

## 8. "왜 문서가 이렇게 많은가" + 동심원 중요도 지도

근거: `workflow diagram for US AF weapon systems.pdf` (대형 관계도) 판독.

**왜 많은가**: 문서가 많은 게 아니라, "측정 하나의 신뢰성"을 떠받치는 기둥이 많은 것.
토크렌치 1개가 정확하려면 → 교정표준·절차서(TO)·교범작성규격·실험실품질(ISO)·시설기준·
인력자격(CFETP)·기록양식(성적서)이 **전부** 보증돼야 함. 문서 수 = 보증할 것의 수.

**전부 커버할 필요 없음. 동심원으로 본다:**
```
바깥 ☆ 참조만 : 기술교범 작성형식(MIL-PRF/S1000D), TO 번호체계, 시설기준, 정보보안
  중간 ★★ 인터페이스 : CFETP(사람), 33K(절차), 성적서·CMRS 양식
    핵심 ★★★ 본체 : 21-113(정신) · TO 00-20-14(절차) · ISO 10012/17025(품질) · 교정주기·소급성 데이터
```
- 다이어그램의 빨간 테두리 박스 = 이미 누가 "중요" 표시: DAFMAN 21-113, CFETP 2P0X1, DAFI 21-101, ISO 17025/10012
- **결정적 통찰**: 군 문서(좌·중앙)가 결국 ISO 10012/17025(우)로 수렴. → 군 체계 전부 베낄 필요 없이
  ISO 공통 뼈대만 잡으면 군 요구의 80% 충족. **이건 CaliBoard가 이미 하는 영역.**
- 집중 3블록: **AF METCAL Policy(노랑) + CMS/CMR(주황) + ISO Standards(초록)**.
  핵심 축 = `21-113 → TO 00-20-14 → 33K → ISO 10012/17025`
